# Страница настройки игры

**Адрес**: /games/{id}/settings

**Описание**: Страница, содержащая форму для настройки данных игры.

## Требования
* К данной странице может получить доступ только автор игры.
* Список настроек:
  * участники;
  * название игры;
  * описание игры;
  * обложка игры;
  * компетенции;
  * файлы игры;
  * удаление игры.
